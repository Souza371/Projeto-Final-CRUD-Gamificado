Atualização Automática

Comandos para configurar seu projeto com hot reload e desenvolvimento eficiente

Comandos para Desenvolvimento

  • npm install -g nodemon
  • npm install --save-dev nodemon
  • npm install -g live-server

Configurar package.json

Adicione estes scripts ao seu package.json:

  • "dev": "nodemon server.js"
  • "start": "node server.js"
  • "client": "live-server --port=3000"

Ferramentas Úteis

1

Nodemon

Reinicia automaticamente o servidor quando detecta mudanças nos arquivos.

2

Live Server

Servidor de desenvolvimento com recarregamento automático para páginas estáticas.

3

Concurrently

Executa múltiplos comandos simultaneamente (client e server).

  • npm install -g concurrently
  • "dev": "concurrently \"npm run server\" \"npm run client\""
